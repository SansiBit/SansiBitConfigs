## 简介

GitIgnore 是 SansiBitConfigs 的子项目之一，为 Git 仓库提供通用的 `.gitignore` 配
置。项目借鉴了 [GitHub/GitIgnore](https://github.com/github/gitignore)，在其基础
上进行精简，也加入了一些未被收录的配置项。

## 使用方法

把 `TARGET.gitignore` 复制到 Git 仓库根目录，并重命名成 `.gitignore`。
